Exercise - Add and delete users in Azure Active Directory - Learn

cwatson-cat
3-4 minutes
  • 7 minutes

You need to add member user accounts for the new developer team in your organization.

In this exercise, you'll create a new Azure Active Directory (Azure AD) organization to hold all of your user accounts. You'll also create a user account, delete a user account, and learn how to recover a deleted user account.

To hold all of the users you create in this exercise, create a new organization.

  1. Sign in to the Azure portal with the same account you used to activate the sandbox.

  2. In the left navigation pane, select Create a resource.

  3. In the Azure Marketplace, search for Azure Active Directory.

  4. Select Create.

  5. On the Create tenant pane, enter the following values:

    • Organization name: Contoso Marketing Company
    • Initial domain name: contosomarketingXXXX where you replace XXXX with numbers or letters to make your domain name unique

  6. Select Create.

  7. Wait for the creation of your tenant to be completed.

  8. Go to the new tenant by selecting Click here to manage your new tenant.

For Azure AD Premium to complete all the exercises in this module, you'll need to activate a free trial.

  1. In the organization you just created, select Getting started.

  2. Under Getting started with Azure AD, select Get a free trial for Azure AD Premium.

  3. Under Azure AD Premium 2, select Free trial > Activate. If you don't have that option, that's ok. You can complete most of the exercises without it.

  4. In the left pane, select Overview.

  5. Refresh the browser until you see Azure AD Premium P2 under the organization name. It may take a couple of minutes.

    You should also see your role listed as the Global administrator.

  6. Copy the Primary domain name to use in the next section.

Add a new user

Now, let's create a user account.

  1. In the Azure AD organization you created, under Manage, select Users > New User.

  2. The User pane now appears. Enter the following values:

    • User Name: chris@contosomarketingXXXXXX.onmicrosoft.com. The domain name should match the primary domain you copied in the previous section.
    • Name: Chris Green

  3. Select Show Password, and copy it somewhere you can refer to it later.

  4. Select Create. The user is now created and registered to your organization.

Delete a user

You can delete users after they're created.

  1. In your Azure AD organization, under Manage, select Users.
  2. From the list, select Chris Green.
  3. Select Delete user. If you don't see that option, select More.
  4. When you're asked to confirm your deletion, select OK.

Recover a deleted item

You can restore deleted users. View the list of the deleted users, and then restore one.

  1. In your Azure AD organization, under Manage, select Users > Deleted users.
  2. You now see all of the users that were deleted within the last 30 days.
  3. Select Chris Green and Restore user.
  4. To confirm, select OK.
  5. Verify that Chris Green's account is recovered by selecting All users.
